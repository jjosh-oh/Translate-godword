==========================================================
  설교 실시간 영어 동시통역 프로그램 — 기술 구성 정리
==========================================================

■ 전체 동작 흐름
----------------------------------------------------------
설교 마이크
  → 브라우저(오디오 캡처, 16kHz PCM 변환)
  → WebSocket 전송
  → 번역 서버(Flask, Python)
  → Google Speech-to-Text (실시간 음성→텍스트)
  → Claude API (텍스트 번역)
  → 송출(브로드캐스트)
  → ① 빔프로젝터 송출 화면  ② 교인 셀폰(ngrok 고정주소)


■ 사용 언어 (프로그래밍)
----------------------------------------------------------
- Python            : 번역 서버(백엔드) 핵심
- JavaScript        : 브라우저 화면 동작(대시보드/송출/셀폰)
- HTML / CSS        : 화면 구성과 디자인
- PowerShell / Batch: Windows 실행·백업 자동화 스크립트


■ 핵심 API / 클라우드 서비스
----------------------------------------------------------
- Anthropic Claude API
    · 모델: claude-opus-4-8 (번역)
    · 프롬프트 캐싱으로 원고 비용 절감
    · 사이트: console.anthropic.com

- Google Cloud Speech-to-Text (음성인식)
    · 스트리밍 인식, 모델 latest_long + enhanced
    · 설교원고/용어집을 인식 힌트(speech adaptation)로 사용
    · 사이트: console.cloud.google.com
    · 인증: gcloud ADC 로그인 방식(서비스 키 없이)


■ 외부 접속(터널) / 배포
----------------------------------------------------------
- ngrok            : 셀폰 고정 주소(QR 재사용)
    · 주소: linguist-copartner-overflow.ngrok-free.dev
    · 사이트: dashboard.ngrok.com
- cloudflared      : 초기 임시 주소 방식(현재는 ngrok 우선)
- QR 이미지 생성    : api.qrserver.com (실패 시 quickchart.io)


■ 설치한 프로그램 / 도구
----------------------------------------------------------
- Python + pip      : 실행 환경 및 패키지 설치
- Google Cloud CLI  : gcloud 인증(application-default login)
- ngrok             : 고정 주소 터널
- cloudflared       : 임시 터널(예비)
- winget            : 위 도구들 설치에 사용
- Chrome 브라우저    : 운영자/송출/셀폰 화면 실행


■ Python 라이브러리 (requirements)
----------------------------------------------------------
- anthropic          : Claude API SDK
- google-cloud-speech: Google 음성인식 SDK
- flask              : 웹 서버
- flask-sock         : WebSocket(오디오/자막 실시간 전송)
- python-dotenv      : API 키(.env) 관리
- pypdf              : PDF 원고 읽기
- python-docx        : Word(docx) 원고 읽기


■ 주요 기능
----------------------------------------------------------
- 실시간 스트리밍 음성인식 + 번역
- 운영자 대시보드 / 빔프로젝터 송출 / 셀폰 화면
- 셀폰 고정 QR 주소(매주 재사용)
- 설교 원고 업로드(번역 참고 + 캐싱)
- 교회 용어집(성경 인명·지명, 인식 정확도 향상)
- 글자색/배경색/폰트 크기 조절, 자막 스크롤
- 원클릭 실행 / 백업 스크립트


■ 참고한 웹사이트 정리
----------------------------------------------------------
- console.anthropic.com       : Claude API 키 발급
- console.cloud.google.com    : 음성인식 API/인증
- dashboard.ngrok.com         : 고정 주소(터널)
- api.qrserver.com            : QR 코드 이미지
