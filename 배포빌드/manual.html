1. 프로그램 소개

ChurchInterpreter는 설교자의 한국어 음성을 실시간으로 영어(또는 다른 언어)로 번역하여 빔프로젝터와 교인 스마트폰에 동시에 자막으로 보여주는 프로그램입니다.

기능설명
실시간 통역설교자 말이 끝나는 즉시 번역 자막 표시
빔프로젝터 송출글자 크기·색상 조절 가능한 전용 화면
셀폰 접속QR코드로 교인 스마트폰에서 자막 확인
설교 원고 활용PDF/Word 원고 업로드 시 번역 정확도 향상
다국어 지원영어, 일본어, 중국어, 스페인어 등

2. 설치 방법

1설치 파일 실행

ChurchInterpreter_Setup.exe를 더블클릭하여 실행합니다.

"Windows의 PC 보호" 경고가 뜨면 추가 정보 → 실행을 클릭하세요.

2설치 경로 선택

기본 경로(C:\Program Files\ChurchInterpreter)를 그대로 사용하거나 원하는 위치로 변경 후 설치를 클릭합니다.

3설치 완료

바탕화면에 ChurchInterpreter 아이콘이 생성됩니다.

⚠ 설치 후 처음 실행 시 API 키를 입력해야 합니다. 아래 3장을 먼저 읽고 키를 준비해 두세요.

3. API 키 발급 (최초 1회)

이 프로그램은 두 가지 외부 서비스를 사용합니다. 각 서비스에서 키를 발급받아야 합니다.

① Anthropic Claude API 키 (번역용)

1사이트 접속 및 회원가입

https://console.anthropic.com 접속
오른쪽 위 Sign Up 클릭 → 이메일 주소 입력 → 인증 메일 확인 → 비밀번호 설정
이미 계정이 있으면 Log In으로 로그인

2결제 수단 등록

로그인 후 왼쪽 메뉴에서 Billing 클릭
→ Add payment method → 신용카드 번호·유효기간·CVC 입력 후 저장
→ Buy credits 클릭 → 금액 입력 후 결제

💡 처음에는 $5~10 정도 충전하면 충분합니다. 월 1회 1시간 예배 기준 약 $2~5 사용됩니다.

3API 키 생성

왼쪽 메뉴 API Keys 클릭
→ 오른쪽 위 + Create Key 클릭
→ 키 이름 입력 (예: church-interpreter) → Create Key 클릭
→ 화면에 나타난 키(sk-ant-api03-...)를 반드시 복사해 메모장에 저장
(창을 닫으면 다시 볼 수 없습니다)

⚠ API 키는 비밀번호와 같습니다. 타인에게 공유하거나 이메일로 보내지 마세요. 유출 시 즉시 해당 키를 삭제하고 새로 발급하세요.

② Google Cloud 음성인식 키

1Google Cloud 접속 및 프로젝트 생성

https://console.cloud.google.com 접속 (Google 계정으로 로그인)
→ 상단 바의 프로젝트 선택 드롭다운 클릭 (처음엔 "프로젝트 선택" 또는 기존 프로젝트명 표시)
→ 팝업 오른쪽 위 새 프로젝트 클릭
→ 프로젝트 이름 입력 (예: church-interpreter) → 만들기 클릭
→ 생성 완료 후 해당 프로젝트가 선택된 상태인지 확인

2결제 계정 연결

왼쪽 상단 메뉴(☰) → 결제 클릭
→ 결제 계정 연결 → 신용카드 등록
(월 사용량이 일정 한도 이하면 무료입니다. 월 60분 이내 음성인식은 무료 범위 내에 들어올 수 있습니다.)

3Speech-to-Text API 활성화

상단 검색창에 Cloud Speech-to-Text API 입력 후 검색 결과 클릭
→ 파란 사용 버튼 클릭 (이미 활성화된 경우 "API 사용 설정됨" 표시)

4서비스 계정 만들기

왼쪽 상단 메뉴(☰) → IAM 및 관리자 → 서비스 계정 클릭
→ 상단 + 서비스 계정 만들기 클릭
→ 서비스 계정 이름 입력 (예: church-stt) → 만들고 계속하기 클릭
→ 역할 선택: Cloud Speech 클라이언트 검색 후 선택 → 계속 → 완료

5JSON 키 파일 다운로드

서비스 계정 목록에서 방금 만든 계정(church-stt@...) 클릭
→ 상단 탭에서 키 클릭
→ 키 추가 → 새 키 만들기 클릭
→ 키 유형 JSON 선택 → 만들기 클릭
→ .json 파일이 자동으로 다운로드됩니다 — 이 파일을 보관합니다

⚠ JSON 키 파일은 비밀번호와 같습니다. 이메일로 보내거나 외부에 공유하지 마세요. 분실 시 기존 키를 삭제하고 새로 발급하면 됩니다.

③ ngrok 계정 (셀폰 접속용)

1ngrok 가입

https://dashboard.ngrok.com → Sign Up (무료)

2Authtoken 복사

로그인 후 Getting Started → Your Authtoken에서 토큰 복사

3(선택) 고정 도메인 발급

Domains 메뉴 → New Domain
무료 고정 도메인 1개 발급 (매주 같은 QR코드 재사용 가능)

4. 처음 실행 — 설정 화면

1프로그램 실행

바탕화면 ChurchInterpreter 더블클릭
검은 콘솔 창이 열리면서 서버가 시작됩니다. (닫지 마세요)

2설정 화면 열기

자동으로 브라우저가 열리거나, 주소창에 입력:
http://localhost:5000/setup

3Claude API 키 입력

3장에서 복사한 sk-ant-... 키를 입력 후 연결 테스트

4Google 키 파일 업로드

파일 선택 버튼으로 다운로드한 .json 파일 선택 후 연결 테스트

5ngrok 설정

Authtoken 입력, 고정 도메인이 있으면 입력

6설정 저장

설정 저장 클릭 → 운영자 대시보드 열기 클릭

✓ 설정은 프로그램 폴더의 .env 파일에 저장됩니다. 다음 실행부터는 자동 적용됩니다.

5. 매주 사용 방법

순서할 일방법
예배 전프로그램 실행바탕화면 ChurchInterpreter 더블클릭
예배 전운영자 대시보드 열기http://localhost:5000/operator
예배 전설교 원고 업로드 (선택)대시보드 좌측 "원고 업로드" → PDF/Word 선택
예배 전빔프로젝터 화면 열기http://localhost:5000 → 더블클릭으로 전체화면
예배 전셀폰 QR 안내대시보드 우측 QR코드를 주보에 인쇄하거나 화면에 표시
예배 시작마이크 켜기대시보드 "🎤 마이크 켜기" 버튼
예배 중자막 확인빔프로젝터 화면 및 셀폰에서 실시간 자막 확인
예배 후마이크 끄기대시보드 "⏹ 마이크 끄기" 버튼
예배 후프로그램 종료콘솔 창 닫기 또는 Ctrl+C

6. 화면 구성 안내

운영자 대시보드 (/operator)

  • 마이크 켜기/끄기 — 통역 시작·중지
  • 언어 선택 — 원본/번역 언어 변경
  • 원고 업로드 — PDF 또는 Word 파일로 번역 정확도 향상
  • 화면 설정 — 글자 크기·색상·배경색 조절 (빔프로젝터에 실시간 반영)
  • QR코드 — 교인 셀폰 접속용 주소

빔프로젝터 화면 (/)

  • 더블클릭 또는 F11 — 전체화면 전환
  • 자막은 아래에서 위로 쌓이며 화면이 넘치면 위에서부터 사라짐

교인 셀폰 화면 (/mobile)

  • QR코드 스캔으로 접속
  • 화면이 꺼지지 않도록 자동 잠금 방지 기능 포함

7. 자주 묻는 질문

Q. 번역이 너무 느려요

인터넷 연결 속도와 API 서버 상태에 따라 0.5~3초 지연이 발생할 수 있습니다. 유선 인터넷 사용을 권장합니다.

Q. 고유명사(사람 이름, 성경 지명)가 잘못 인식돼요

대시보드 → 용어집 업로드에서 교회 특유의 고유명사를 추가하면 인식 정확도가 올라갑니다.

Q. 셀폰에서 QR을 찍었는데 접속이 안 돼요

ngrok 설정이 완료되었는지 확인하세요. 대시보드의 QR코드 영역에 주소가 표시되어 있어야 합니다.

Q. 검은 콘솔 창을 실수로 닫았어요

바탕화면 아이콘을 다시 더블클릭해 프로그램을 재시작하세요.

Q. API 키를 바꾸고 싶어요

브라우저에서 http://localhost:5000/setup에 접속해 새 키를 입력 후 저장합니다.

Q. 번역 비용이 얼마나 들어요?

월 1회 1시간 예배 기준 약 $2~5 수준입니다. Anthropic 콘솔에서 실시간 사용량을 확인할 수 있습니다.
