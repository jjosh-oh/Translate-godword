처음 사용 시 API 키를 설정해 주세요

💬 설정이 처음이신가요?

무료 Claude(claude.ai)가 아래 3가지 발급을 하나씩 안내해 드립니다. 개발 지식이 없어도 대화하며 따라 할 수 있어요.

① 질문 복사 → ② Claude 열기 → ③ 붙여넣기(Ctrl+V) → 안내에 따라 진행
1. Claude API 키 Anthropic
발급: console.anthropic.com → API Keys
✓ 이미 저장된 키가 있습니다 (변경하려면 새 키 입력)
2. Google Cloud 인증 Speech-to-Text
발급: console.cloud.google.com → 서비스 계정 → 키 생성
✓ 이미 google-key.json이 있습니다 (교체하려면 새 파일 선택)
3. ngrok 터널 셀폰 QR

교인 셀폰이 QR코드로 접속하려면 ngrok이 필요합니다.
무료 계정으로 발급한 Authtoken을 입력해 주세요.

발급: dashboard.ngrok.com → Your Authtoken
✓ 이미 Authtoken이 저장되어 있습니다
ngrok 대시보드 → Domains → 무료 도메인 1개 발급 후 입력 (비워도 됨)
→ 운영자 대시보드 열기
