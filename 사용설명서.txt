==========================================================
   설교 실시간 번역 시스템 — 사용설명서
==========================================================

■ 매번 사용하는 방법 (가장 중요)
----------------------------------------------------------
1. C:\Claude 폴더에서 "실행-셀폰포함.bat" 더블클릭
2. 잠시 기다리면:
   - 검은 서버 창이 뜨고
   - Chrome에 운영자 대시보드가 자동으로 열립니다
3. 운영자 대시보드(http://localhost:5000/operator)에서:
   - 마이크 장치 선택
   - 언어 설정 확인 (예: 한국어 → 영어)
   - "송출 창 열기" 버튼 → 빔프로젝터용 자막 화면이 열림
   - "마이크 시작" 버튼 → 통역 시작
4. 셀폰 접속:
   - 대시보드 "셀폰 접속(QR)" 칸에서 "🔄 주소 불러오기" 클릭
   - 생성된 QR을 교인들이 스캔 → 셀폰에서 자막 표시

■ 종료 방법
----------------------------------------------------------
- 검은 창들(서버, 터널)을 닫으면 종료됩니다.
- 예배 중에는 절대 닫지 마세요. 닫으면 자막/셀폰 접속이 끊깁니다.

■ 주의사항
----------------------------------------------------------
- 음성인식은 Chrome 브라우저에서만 동작합니다.
- 설교 마이크(Vocaster 등)를 Windows 기본 입력 장치로 두면 가장 안정적입니다.
- PC는 가능하면 유선 인터넷(랜) 권장.
- 셀폰 접속 주소는 실행할 때마다 새로 바뀝니다 (정상).

■ 비용 (예배 1시간 기준 대략)
----------------------------------------------------------
- 음성인식(Google Speech): 약 $1.4 (신규 가입 $300 크레딧으로 한동안 무료)
- 번역(Claude): 약 $1 내외
- 셀폰 외부접속(cloudflared): 무료

■ 셀폰 고정 주소 (바뀌지 않음)
----------------------------------------------------------
- https://linguist-copartner-overflow.ngrok-free.dev/mobile
- 이 주소로 QR을 한 번만 만들어 두면 매주 재사용 가능합니다.
- 대시보드 "셀폰 접속(QR)" 칸에서 "주소 불러오기" → "QR 생성"

■ 자주 막히는 부분
----------------------------------------------------------
- 대시보드가 안 열리면: Chrome 주소창에 직접 http://localhost:5000/operator 입력
- 셀폰 주소가 안 뜨면: 대시보드 "주소 불러오기" 버튼 클릭
- 한글 인식이 안 되면: Windows 소리 설정에서 설교 마이크를 기본 입력으로 지정

■ 파일 설명
----------------------------------------------------------
- 실행-셀폰포함.bat : 전체 시작 (이것만 실행하면 됨)
- server.py         : 번역 서버 (수정 금지)
- operator.html     : 운영자 대시보드 화면
- display.html      : 빔프로젝터 송출 화면
- mobile.html       : 셀폰 화면
- .env              : API 키 (외부에 절대 공유 금지)
- ngrok-domain.txt  : 셀폰 고정 도메인 설정값
- ngrok설정.bat     : (최초 1회 완료됨) ngrok 토큰/도메인 설정
- 백업.bat          : 전체 폴더를 백업 폴더에 복사

■ 백업 / 다른 PC로 옮기기
----------------------------------------------------------
- "백업.bat"을 실행하면 C:\Claude-백업\(날짜) 폴더에 전체가 복사됩니다.
- 다른 PC로 옮길 때는 C:\Claude 폴더 전체를 복사하고,
  그 PC에서도 Python / gcloud 로그인 / pip install 을 한 번 해주면 됩니다.
